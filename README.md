# Khana Khazana

## Description
Khana Khazana is a web application designed to provide users with a variety of delicious recipes and cooking tips. The app allows users to browse through recipes, filter by ingredients, and save their favorites.

## Features
- Browse a wide variety of recipes.
- Search and filter recipes by ingredients.
- Save favorite recipes for quick access.
- User-friendly interface for easy navigation.

## Installation
To run the application locally, clone the repository and install the necessary dependencies:
```bash
git clone https://github.com/avantika2195/Khana-Khazana.git
cd Khana-Khazana
npm install
npm start
